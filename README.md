# AI-Hedge-Fund-Simulator
Develop an AI-powered automated trading system that constructs and manages investment portfolios that incorporate customizable risk preferences, market sentiments, and macro environment changes, instead of simple techinal analysis.
The system analyzes market data to generate trading decisions and provides explanations for each trade and portfolio adjustment, helping users understand the reasoning behind each strategy.
The project simulates how a quantitative hedge fund research pipeline works:
data collection → multi-agent analysis → portfolio optimization → backtesting → visualization.

## Key Features
1. Portfolio Construction <br>
Asset Types: stocks, bonds, ETFs. <br>
Risk Management: portfolio diversification, volatility control, position sizing.
2. Customizable Investment Strategy <br>
Users can define investment preferences themselves, including:
- asset selection
- order types
- risk tolerance
- portfolio size
- investment horizon <br>
AI will then generate an optimal strategy based on these constraints.
3. Explainable AI Trading Decisions <br>
Every trade generated by the system will include explanations. 

## System Architecture
Raw Data -> Research Agents -> LLM Aggregator -> Decision Layer -> Backtesting Engine -> Visualization Dashboard

### Data Sources
1. Market Data
2. News & Sentiment
3. Macroeconomic Data
4. Company Fundamentals

### AI Agents
1. Techical Analysis Agent
2. Sentiment Analysis Agent
3. Historical Trend Agent
4. Fundamental Analysis Agent
5. Macroeconomic Agent

### LLM Aggregator / Decision Layer
1. Integrate signals from all agents
2. Enable customizable decisions from users

### Results
1. The system's portfolio performance VS. Indices
2. Trading history and strategy reports


## Technology Stack
- Backend: Python, FastAPI
- Data Processing: pandas, NumPy
- Machine Learning: PyTorch, scikit-learn
- Financial Analysis: QuantLib, PyPortfolioOpt
- Frontend: React, D3.js / Chart.js
